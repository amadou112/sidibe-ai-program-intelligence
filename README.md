## Overview

The **Sidibe Enterprises AI Program Intelligence Platform** is a modern enterprise application built to demonstrate how Artificial Intelligence can improve Program and Project Management.

The platform enables Program Managers to upload project documentation, analyze project health, identify risks, generate executive summaries, and interact with project information through an AI-powered interface.

This project showcases enterprise application architecture, modern frontend development, cloud deployment, and AI integration.

## Getting Started

Clone the repository

```bash
git clone https://github.com/amadou112/sidibe-ai-program-intelligence.git
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Open your browser

```
http://localhost:3000
```
